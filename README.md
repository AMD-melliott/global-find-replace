# Global Find & Replace

## Exercise Goal
Learn to use VS Code's powerful search and replace features across an entire project.

## Scenario
The product name has changed from "Project Alpha" to "Omega App". You need to update all documentation files.

## Instructions
1. Clone this repository
2. Open the Search sidebar (magnifying glass icon)
3. Search for "Project Alpha" across all files
4. Preview the matches
5. Use the Replace field to enter "Omega App"
6. Click through results to review changes before applying
7. Execute "Replace All"
8. Go to Source Control - you should see multiple modified files
9. Stage all changes at once
10. Commit with message: "Refactor product name from Project Alpha to Omega App"

## Skills Practiced
- Global search across files
- Preview changes before applying
- Batch commits
- Managing multi-file changes
